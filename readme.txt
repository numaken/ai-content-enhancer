AI で本文を整え、バックアップ/復元ができる編集支援プラグイン。

== Installation ==
1. `ai-content-enhancer` フォルダごと `wp-content/plugins/` に設置
2. 管理画面 > プラグイン で有効化
3. 設定 > AI Content Enhancer から各種設定
